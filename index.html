Simple Setup Interface

No Technical Skills Required Our interactive setup wizard guides you through everything. Most users are up and running in under 5 minutes.
# Step 1: Install from npm (requires Node.js 16+)
$ npm install -g aicw

# Step 2: Set up your API key (get free key from openrouter.ai)
$ aicw setup

# Step 3: Start the interactive interface
$ aicw

# Step 4: View your reports in the browser
$ aicw serve

The interactive mode will guide you through creating your first monitoring project.

Technical Details

How It Works

  1. Define Questions: Create a list of questions you want to monitor
  2. Query AI Models: Automatically fetch responses from multiple models
  3. Extract Entities: AI identifies keywords, organizations, events, places
  4. Calculate Metrics: Generate influence scores and track trends
  5. Generate Reports: Interactive HTML with network graphs and data tables

Architecture

  • Language: TypeScript/Node.js
  • API: OpenRouter (unified access to all models)
  • Storage: Local JSON files (no database needed)
  • Reports: Static HTML + JavaScript
  • Dependencies: Minimal, no frameworks required

Command Reference

# Full pipeline
$ aicw all ProjectName

# Individual steps
$ aicw prepare ProjectName # Convert questions to structured format
$ aicw fetch ProjectName # Query AI models
$ aicw compile ProjectName # Extract entities with AI
$ aicw enrich ProjectName # Calculate metrics and trends
$ aicw report ProjectName # Generate HTML report

# Analysis
$ aicw compare ProjectName 2024-01-01 2024-02-01 # Compare two dates
$ aicw compare ProjectName trends # Full trend analysis
